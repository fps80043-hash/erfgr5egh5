❓ Справка (Render)

        
    
          
  1. Залей этот проект на GitHub.
    2. 
          
  2. На Render создай New → Web Service, выбери репозиторий.
    3. 
          
  3. Build command: pip install -r requirements.txt
    4. 
          
  4. Start command: uvicorn app:app --host 0.0.0.0 --port $PORT
    5. 
          
  5. Если хочешь Groq — добавь Environment Variable: GROQ_API_KEY.
    6. 
        

        
Важно: free-tier Render может “засыпать” без трафика — первый запрос после простоя иногда медленный.
