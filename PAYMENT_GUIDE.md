## 💳 Платёжные системы: кто примет и что нужно

### 1. **ЮKassa (бывш. Яндекс.Касса)** ⭐ Рекомендую

**Статус:** Нужно ИП/ООО

**Плюсы:**
- ✅ Самая популярная в России
- ✅ Принимает карты, ЮMoney, QIWI, СБП
- ✅ Низкие комиссии (от 2.8%)
- ✅ Хорошая техподдержка

**Минусы:**
- ❌ Нужно ИП/ООО
- ❌ Проверка сайта (могут отказать за игровую валюту)

**Что нужно сделать:**
1. Зарегистрировать ИП
2. Открыть счёт в банке
3. Оформить как "Цифровые товары" или "Онлайн-услуги"
4. Подключить онлайн-кассу
5. Добавить все юридические страницы на сайт

**Как подключить:**
```python
# Установка
pip install yookassa

# Пример кода
from yookassa import Configuration, Payment

Configuration.account_id = 'ваш_shop_id'
Configuration.secret_key = 'ваш_secret_key'

# Создание платежа
payment = Payment.create({
    "amount": {
        "value": "100.00",
        "currency": "RUB"
    },
    "confirmation": {
        "type": "redirect",
        "return_url": "https://your-site.com/payment/success"
    },
    "capture": True,
    "description": "Robux 100"
})
```

**Комиссия:** 2.8% + 10₽ за транзакцию

### 5. **Cryptomus** ✅ Доступно сейчас

**Статус:** Можно БЕЗ ИП

**Плюсы:**
- ✅ Аналог CryptoBot
- ✅ Криптовалюта
- ✅ Не нужно ИП
- ✅ Низкие комиссии

**Минусы:**
- ⚠️ Только крипта (не для всех)

**Как подключить:**
```python
# API похоже на CryptoBot
import requests

def create_invoice(amount):
    url = "https://api.cryptomus.com/v1/payment"
    headers = {
        "merchant": "your_merchant_id",
        "sign": "your_sign"
    }
    data = {
        "amount": str(amount),
        "currency": "USD",
        "order_id": "order_123"
    }
    response = requests.post(url, headers=headers, json=data)
    return response.json()
```

**Комиссия:** 0.5% - 1%
