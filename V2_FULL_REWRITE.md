## Единый вход v1 ↔ v2

Работает через существующий cookie-механизм бэкенда — `SESSION_COOKIE` выставляется с `path="/"`, значит и `/` (v1), и `/v2` делят одну сессию. Фронтенд везде использует `credentials:'same-origin'`. Никаких изменений в бэке для этого не нужно, просто:

- Залогинился на v1 → переходишь на /v2 → ты уже залогинен.
- Залогинился на v2 → переходишь на / → ты уже залогинен.
- `doLogout()` вызывает `/api/auth/logout` → выходишь везде одновременно.

## Новая капча — drag-to-confirm

Убран ввод чисел (было: «введите сумму 5+3»). Теперь — слайдер, который надо провести до конца.

**Как работает:**
1. Фронт запрашивает `/api/captcha/challenge?purpose=login` — получает `{token, a, b}`.
2. Фронт локально вычисляет `answer = a+b` и сохраняет.
3. Юзер перетаскивает слайдер (мышь или палец) — при достижении 95% позиции состояние становится `verified`.
4. При отправке формы передаются `captcha_token` и `captcha_answer` — бэковый `_captcha_verify` принимает их без изменений.

**Тонкости:**
- Слайдер работает на мыши (`mousedown/move/up`) И на тач-экранах (`touchstart/move/end`).
- При возврате слайдера ниже 95% — сброс. Это UX-защита от случайных кликов.
- Капча-токен привязывается к purpose (`login`, `register`, `reset`), как раньше.
- Безопасность: бот может вычислить `a+b` так же, как юзер, — для реальной защиты нужно Turnstile/hCaptcha.

## Полный перечень функционала

### Главная (`/v2`)
- Hero с CTA к Robux и Магазину.
- Блок «Возможности» (4 карточки).
- **Блок отзывов** (6 последних с аватарами и звёздами).
- FAQ аккордеон (5 вопросов, один открывается).
- Футер со ссылками на разделы, документы, Telegram.

### Купить Robux (`/v2/robux`)
- Сток через `/api/robux/stock`.
- Калькулятор с котировкой `/api/robux/quote` — обновляется на вводе (дебаунс 300мс).
- Quick-amounts: 50, 100, 250, 500, 1000.
- Переключатель «Обычный/Авто» (авто только для Premium, иначе заблокирован).
- Режимы: **по нику** и **по ссылке на геймпасс**.
- **Проверка** (`/api/robux/inspect`) с показом карточки геймпасса (картинка, название, ID, цена).
- **Покупка** (`/api/robux/order_create`) — сначала проверка баланса, при нехватке открывается топап.
- После покупки редирект на v1 со ссылкой на заказ (`/#robux-order-X`), т.к. просмотр статуса заказа использует WebSocket и лучше работает в v1.

### Магазин (`/v2/shop` + `/v2/g/<id>`)
- Табы категорий (чипы).
- Если категория пустая и выбраны «Все» — автоматически показываются карточки категорий (красивая навигация).
- Поиск работает по всем товарам (из шапки).
- Детальная страница категории с под-категориями (табы) и баннером.
- Карточки с обложками, бейджами, «нет в наличии».

### Профиль (`/v2/profile`)
- Гость-экран со входом.
- Блок аватара с загрузкой (`/api/user/avatar` multipart) и сбросом (`/api/user/avatar/reset`).
- Бейджи Premium и Admin.
- Стата: баланс, Premium-срок.
- Быстрые действия: Пополнить, Поддержка, AI инструменты, Кейсы, Выйти.
- Табы: **Обзор** (пополнено/потрачено/покупок из `/api/balance/history` + `/api/purchases`), **Покупки** (список), **Безопасность** (ссылка на v1 для смены пароля).

### Отзывы (`/v2/reviews`)
- Полный список через `/api/reviews`.
- Кнопка «Оставить отзыв» — проверяет, не оставлял ли юзер уже (`/api/reviews/my`), показывает форму с 5-звёздной оценкой.
- Отправка через `/api/reviews/create`.

### Модалки

**Авторизация**
- Вход, Регистрация, 2FA-подтверждение, подтверждение email.
- Все — в одном модале, переключаются по steps.
- Слайдер-капча.

**Товар** — карточка с обложкой, описанием, ценой, кнопкой покупки.

**Пополнение** — 6 паков (100–5000 ₽), кастомная сумма, табы крипта/промокод. Автополлинг статуса платежа каждые 5 сек в течение 15 мин.

**Кейсы + Инвентарь**
- Список кейсов (`/api/cases/config`), свободный/платные/money.
- Полноценная рулетка с 48 анимированными элементами и CSS transition 5.2с.
- Стилизация призов по tier (1–4).
- Вкладка инвентаря с Use (`/api/inventory/use`) и Delete (`/api/inventory/delete`).

**AI инструменты**
- **AI-чат** через `/api/tools/ai_chat` — пузырьки user/bot, сохраняется в памяти сессии.
- **Генератор описаний** — сначала проверка cookie через `/api/roblox/check_cookie`, затем форма параметров и генерация через `/api/tools/generate_description` с кнопкой копирования.

**Поддержка**
- Список тикетов (`/api/support/list`).
- Открытие тикета (`/api/support/messages?ticket_id=X`) — стилизованный чат с пузырьками админа/юзера.
- Отправка ответа (`/api/support/reply`).
- Создание нового тикета (тема + категория из 8 значений + текст).

**Админка** (только для `is_admin`)
- Закрытый оверлей (z-index 600), не модалка — полноэкранный.
- Табы: Игры (категории) / Товары.
- Списки с iconbtn для редактирования и удаления.
- Редактор категорий и товаров с ВСЕМИ полями: title, id, sort, visible, banner_url, banner_color, субкатегории, price, badge, descripton_html и т.д.
- Кнопка загрузки картинки через `/api/admin/upload_banner` (multipart) с превью.
- Сохранение через `/api/admin/shop_config` (POST `{config: SHOP}`).
